Vaardigheden
Illustrator

De meeste functionaliteiten van Illustrator beheers ik. Ik kan zonder problemen tot ontwerpen van logo's, persona's en meer komen.

Adobe XD

In korte tijd kom ik tot uitgewerkte prototypes met verschillende interactiviteit.

Axure

Van lo-fi wireframes tot meer complexe prototypes. Met Axure lukt mij dit allemaal.

Balsamiq

Het werken met Balsamiq doe ik zonder moeite. Ik maak er weinig gebruik van, maar Balsamiq kan soms net de snelle oplossing bieden voor een lo-fi wireframe.

Photoshop

Ik ben bekend met de basisfunctionaliteiten van Photoshop. Meer heb ik voor mijn huidige workflow niet nodig.

HTML/CSS

Ik maak met gemak een website door het gebruik van HTML en CSS. Door deze kennis kan ik als designer ook beter met front-end developers communiceren.

Javascript

Waar nodig gebruik ik op een laag niveau JavaScript. Het voordeel dat kennis van JavaScript mij biedt is wederom de betere communicatie met front-end developers.
